Selectors In CSS!

HTML Attribute Selectors

CSS gives you many ways to target an element in your HTML code. Let's take a look at them! NOTE: I already covered HTML Selectors in my Intro to CSS page! Click here to have a quick review! NOTE: I will be using the p tag
  • Universal Selector p[] - p[class]

    Matches any Element that has attribute. The attribute's value does not matter here

  • Equality Selector p[=] - p[class="bordered"]

    Matches any Element that has an attribute of a specific value

  • Space Selector p[~=] - p[class~="bordered"]

    Matches any Element that has an attribute of a specific value.
    This attribute can appear in a list of attributes - seperated by spaces.

  • Prefix Selector p[^=] - p[class^="bordered"]

    Matches any Element that has an attribute value - that begins with a given string "".

  • SubString Selector p[*=] - p[class*="bordered"]

    Matches any Element that has an attribute value - that contains a given string "".

  • Suffix Selector p[$=] - p[class$="bordered"]

    Matches any Element that has an attribute value - that ends in a given string "".

HTML Pseudo-Elements

Pseudo Elements can also be part of any Selector. They begin with a colon : and allow for even MORE specific Selecting. amazing right? Some examples of pseudo-elements are
:hover
:active
:focus
:first-letter
:first-line
:link
:visited

All of these are great tools to have a know! Have fun styling with CSS! :)
